# 📱 Tech News "Pulse"

## 🚀 Overview
Tech News "Pulse" is a simple news aggregator web application focused on technology and startup news. It fetches real-time data from an API and displays it in a user-friendly interface.



## 🎯 Objective
- Learn API integration using JavaScript (`fetch`)
- Implement search, filtering, and UI features
- Build a responsive and interactive web app



## 🌐 API Used
- https://newsapi.org/



## ✨ Features

### 🔍 Core Features
- Category Filters (AI, Crypto, Gadgets)
- Search functionality
- Save for Later feature
- "Read More" links to full articles
- Image fallback for missing images

### 🌙 Bonus Feature
- Dark Mode toggle
- Saves preference using localStorage



## 🛠️ Technologies Used
- HTML5
- CSS
- JavaScript
- Fetch API
- Local Storage



## ⚙️ How It Works
1. Fetch data from NewsAPI
2. Display articles dynamically
3. Apply search and filters using:
   - map()
   - filter()
   - find()
4. Store user preferences (dark mode, saved articles)
